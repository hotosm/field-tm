What is it?

The Field Tasking Manager (Field-TM) helps teams add local knowledge to map features by coordinating mapping in the field.

Field-TM facilitates collaborative mapping by supporting and extending mature tools like ODK and QField. It is a standalone mobile and web workflow that helps teams collect geospatial data efficiently with commonly available Android devices.

Main Features

Flexible Data Collection

Support diverse survey forms, multiple data formats, and adaptable workflows that fit different operational needs.

Open Source and Customizable

Adapt Field-TM to your own survey logic, data structures, and integration requirements.

Entity-Level Tracking

Monitor survey progress at task and feature level so managers can make informed decisions quickly.

Who is it for?

Project Manager

Create and coordinate field campaigns

Project managers use Field-TM to set up mapping projects, define areas of interest, design survey forms, and split work into manageable tasks.

  1. Define a project area and upload or choose a survey form
  2. Split the area into tasks and generate a data extract
  3. Publish the project to ODK Central or QFieldCloud
  4. Share a QR code so mappers can start collecting data
Field Mapper

Collect data in the field with your phone

Mappers do not need to log in to Field-TM. Once a project manager publishes a project, mappers access it directly in their mobile app of choice.

  1. Scan the QR code shared by your project manager
  2. The project loads automatically in ODK Collect or QField
  3. Navigate to your assigned task area and collect data
  4. Submit your survey, which goes straight to the server

Field-TM Use Cases

1. Community Mapping

Field Tasking Manager empowers communities to map informal settlements, public facilities, and local services for inclusive planning.

2. Urban Planning and Management

Build smarter cities with coordinated surveys of houses, roads, and utilities through clear, manageable mapping tasks.

3. Disaster Response and Recovery

Rapidly map damaged infrastructure after disasters, coordinate volunteers, and deliver reliable data for relief operations.
